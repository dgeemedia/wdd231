– Site Plan

Site Name

9jaVault was chosen to reflect national identity. “9ja” is a popular Nigerian slang for Nigeria, while “Vault” represents a secure place for financial insights and investment comparisons. Together, it resonates with Nigerian users and establishes brand familiarity and trust.

Site Purpose

9jaVault helps users compare daily investment rates—like fixed deposits and treasury bills—offered by Nigerian banks. It features a rate calculator, daily update subscription, and a premium plan for deeper insights. Users can make more informed financial decisions with real-time data tailored for Nigeria’s market.

User Scenarios

  • “How much interest will I earn if I invest ₦500,000 at 13% for 90 days?”
  • “Which bank currently offers the best rate for a 180-day fixed deposit?”

Color Scheme

These colors reflect Nigerian identity and financial trust:

  • Green (#008751): Used for headings, calculator background, and main highlights.
  • Gold (#f0c36d): Used for the navbar and accent buttons.

Typography

Arial is used site-wide for both headings and body text due to its clarity and universal support. It ensures readability across devices and screen sizes.

Wireframe

Mobile View

  • Top: Logo + hamburger nav (Home, Login, About)
  • Hero: Welcome message + Compare Rates button
  • Calculator: Amount, rate, tenure, calculate button
  • Features: List of 3 site benefits
  • Footer: Copyright

Desktop View

  • Top: Logo left, full menu right (Home, Login, About)
  • Hero: Split layout - left (text & CTA), right (calculator)
  • Below hero: Features section centered
  • Footer: Full-width bar with copyright

Validation & Best Practices

  • HTML and CSS validated using W3C tools
  • Alt text used for all meaningful images
  • Responsive layout with viewport meta tag
  • Color contrast meets accessibility standards
  • Semantic tags used for structure: <nav>, <header>, <section>, <footer>
